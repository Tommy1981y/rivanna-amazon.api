Bienvenido a mi Proyecto de Amazon FBA

Este es un bot diseñado para automatizar tareas dentro de Amazon FBA, como la gestión de inventarios, precios y pedidos.

Este bot usa las APIs de Amazon, como Amazon MWS y SP-API, para:

  • Actualizar precios de productos.
  • Gestionar inventarios y niveles de stock.
  • Analizar y responder a reseñas de productos.
  • Automatizar campañas publicitarias (PPC) en Amazon.

Para más detalles, visita el repositorio de GitHub: Mi Repositorio en GitHub
